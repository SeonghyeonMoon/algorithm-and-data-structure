# Intellij

## 장점

1. 범위 확장 등의 유용한 단축키
2. Git, Github 사용 원할
3. Debug 변수 코드에 표시

## 단축키 정리

### Reformat

1. Ctrl + Alt + I - Auto Indent
2. Ctrl + Alt + L - Auto Reformat

### 편집

1. **Ctrl + W - Extend Selection**  
2. **Ctrl + Shift + Enter - Code Completion**
3. Alt + Shift + Up/Down - Move Line Up/Down
4. Shift + F6 - Rename

### Navigate

1. Ctrl + F4 - Close Tab
2. Ctrl + Tab - Switcher
3. **Alt + Left/Right - Switcher**

### Git

1. Ctrl + K - Commit
2. Ctrl + Shift + K - Push
3. Alt + 9 - Git Open

### Run & Debug

1. Shift + F10 - Run
2. Alt + 4 - Run Page
3. Shift + F9 - Debug
4. Alt + 5 - Debug Page
