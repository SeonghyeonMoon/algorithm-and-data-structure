# [백준] 부분합

## 풀이 1

누적합을 통한 풀이

```js
const solution = () => {
  const [n, s] = input().split(' ').map(Number);
  const numbers = input().split(' ').map(Number);

  const cumulatives = numbers.reduce((acc, cur) => {
    acc.push(acc[acc.length - 1] + cur);
    return acc;
  }, [0]);

  for (let charNum = 1; charNum <= n; charNum++) {
    for (let i = 0; i <= n - charNum; i++) {
      if (cumulatives[i + charNum] - cumulatives[i] >= s) {
        return charNum;
      }
    }
  }
  return 0;
}

console.log(solution());
```

## 풀이 2

투 포인터를 통한 풀이로 수정

```js
const solution = () => {
  const [n, s] = input().split(' ').map(Number);
  const numbers = input().split(' ').map(Number);
  let left = 0;
  let right = 0;
  let sum = numbers[0];
  let answer = Infinity;

  while (true) {
    if (sum >= s) {
      answer = Math.min(answer, right - left + 1);
      sum -= numbers[left++];
    } else {
      sum += numbers[++right];
      if (right === n) break
    }
  }
  return answer === Infinity ? 0 : answer;
}

console.log(solution());
```
