# [백준] [센서](https://www.acmicpc.net/problem/2212)

```js
const solution = () => {
  const n = Number(input());
  const k = Number(input());

  const sensors = input()
    .split(' ')
    .map(Number)
    .sort((a, b) => a - b);

  const distances = sensors
    .slice(1)
    .map((sensor, i) => sensor - sensors[i])
    .sort((a, b) => b - a);

  return sensors[n - 1] - sensors[0] - distances.slice(0, k - 1).reduce((a, b) => a + b, 0);
};

console.log(solution());
```
