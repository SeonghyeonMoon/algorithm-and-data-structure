# [이코테] 곱하기 혹은 더하기(p312)

## 구현

```py
answer = 0
for num in input():
    if num == '0' or num == '1' or answer == 0:
        answer += int(num)
    else:
        answer *= int(num)
print(answer)
```

```js
const s = prompt()
answer = 0

for (let num of s) {
  if (answer === 0 || num === '1' || num === '0') {
    answer += parseInt(num)
  } else {
    answer *= parseInt(num)
  }
}

console.log(answer)
```
