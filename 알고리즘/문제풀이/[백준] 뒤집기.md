# [백준] [뒤집기](https://www.acmicpc.net/problem/1439)

## 구현

```py
s = input()
print(min(len([x for x in s.split('1') if x]),len([x for x in s.split('0') if x])))
```

```js
const s = prompt()

oneCount = 0
for (let num of s.split('0')) {
  if (num) {
    oneCount += 1
  }
}

zeroCount = 0
for (let num of s.split('1')) {
  if (num) {
    zeroCount += 1
  }
}

console.log(Math.min(oneCount, zeroCount))
```
