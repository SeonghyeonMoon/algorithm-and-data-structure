# [이코테] 볼링공 고르기(p315)

## 구현

```py
from collections import Counter

n, m = map(int, input().split())
k = list(map(int, input().split()))
counter = Counter(k)
answer = 0
for idx in range(1, m + 1):
    n -= counter[idx]
    answer += n * counter[idx]
print(answer)
```

```js
const balls = [1, 3, 2, 3, 2];
const counter = {};

for (const ball of balls) {
  if (ball in counter) {
    counter[ball] += 1;
  } else {
    counter[ball] = 1;
  }
}

answer = parseInt((n * (n - 1)) / 2);

for (const count of Object.values(counter)) {
  if (count > 1) {
    answer -= parseInt((count * count - 1) / 2);
  }
}

console.log(answer);
```
