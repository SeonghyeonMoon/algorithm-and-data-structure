# [이코테] 모험가 길드(p311)

## 구현

```py
n = int(input())
answer = 0
group = []
for fear in sorted(list(map(int, input().split()))):
    group.append(fear)
    if group[-1] <= len(group):
        answer += 1
        group = []
print(answer)
```

```js
const N = parseInt(prompt())

const fears = prompt().split(' ').map(num => parseInt(num)).sort((a, b) => b - a)

let answer = 0
let groupNum = 0
for (const fear of fears) {
  groupNum += 1
  if (groupNum >= fear) {
    answer += 1
    groupNum = 0
  } 
}

console.log(answer)
```
