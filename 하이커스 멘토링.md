## interface, type 사용

Interface - 선언 병합(Declaratio Merging)이 가능
Typescript팀이 Interface 권장

```ts
type Person = {
  name: string;
  age?: number;
};

interface Developer extends Person {
  skills: string[];
}

type Developer = Person & {
  skills: string[];
};
```
