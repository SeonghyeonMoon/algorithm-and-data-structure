# Bundler

## Parcel vs Webpack

| Parcel                       | Webpack                 |
| ---------------------------- | ----------------------- |
| 구성 없는 단순한 자동 번들링 | 매우 꼼꼼한 구성        |
| 소/중형 프로젝트에 적합      | 중/대형 프로젝트에 적합 |

## loader

css-loader - Javascript에서 css를 해석가능
style-loader - 해석된 css를 html에 삽입
