## 관련 문제 풀이

1. 추석 트래픽(프로그래머스)
   시간문제 초로 환산

```py
def solution(lines):
    times = []
    for line in lines:
        time, during = line.split(' ')[1:]
        h, m, s = time.split(':')
        end = int(h) * 3600 + int(m) * 60 + float(s)
        start = end - float(during[:-1])
        if start < 0: start = 0
        times.append([start, end])
    times.sort(key = lambda x: x[1])
    answer = 1
    for idx, time in enumerate(times[:-1]):
        end = time[1]
        cnt = 1
        for ctime in times[idx + 1:]:
            start = ctime[0]
            if end + 0.999 > start:
                cnt += 1
        answer = max(answer, cnt)
    return answer
```
